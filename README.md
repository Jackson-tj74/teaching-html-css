# 🌐 Teaching HTML & CSS Portfolio Project

This repository was created while I was teaching HTML and CSS to beginners, with simple examples and practical exercises.

## 📌 About the Project
This is a beginner-friendly multi-page portfolio website built using only **HTML and CSS**. It includes basic layout structure, navigation, and responsive sections to help learners understand web development fundamentals.

## 🧑‍💻 Pages Included
- Home (index page)
- About page
- Services page
- Contact page

## 🎯 Features
- Responsive navigation bar
- Hero section with profile image
- Services section with cards layout
- Contact form
- Footer with social links
- Clean and simple CSS styling

## 🛠️ Technologies Used
- HTML5
- CSS3
